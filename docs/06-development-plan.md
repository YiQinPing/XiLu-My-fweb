# 分阶段执行计划

## 总览

| Phase | 名称 | 预估时间 | 核心交付 |
|-------|------|---------|---------|
| Phase 1 | MVP 核心写作系统 | 4-6 周 | 可用的写作编辑器和基础管理 |
| Phase 2 | 关系与叙事结构 | 2-3 周 | 关系图、时间线、伏笔系统 |
| Phase 3 | AI 与数据驱动 | 2-3 周 | AI 助手、灵感、数据统计 |
| Phase 4 | 完善与高级功能 | 2-3 周 | 导出、修订、协作、移动端 |

## 风险与依赖

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| TipTap 中文输入兼容性 | 编辑器核心体验 | 提前验证，准备备选方案(Slate.js) |
| PostgreSQL FTS 中文分词效果 | 搜索体验 | 测试 zhparser，必要时上 Elasticsearch |
| Anthropic API 成本 | Phase 3 预算 | 实现缓存和用量限制，支持 API Key 用户自配 |
| 移动端编辑器体验 | 跨平台可用性 | 提前做移动端编辑器原型 |
