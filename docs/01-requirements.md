# 功能需求规格

## 项目愿景

希陆Flow 是一款面向长篇小说作家的全流程写作辅助平台。覆盖从灵感捕捉、世界观构建、人物设计、大纲规划、正文写作到数据统计、修订编辑的完整创作周期。

**目标用户**：网络小说作者、传统小说作者、剧本创作者

## 模块总览

```
希陆Flow
├── 1. 正文写作       ← 作家80%时间所在
├── 2. 大纲设计       ← 结构规划
├── 3. 世界观设计     ← 架空世界构建
├── 4. 人物设计       ← 角色塑造
├── 5. 人物关系图     ← 关系可视化
├── 6. 伏笔设计       ← 草蛇灰线管理
├── 7. 时间线设计     ← 事件编排
├── 8. 灵感           ← 想法捕捉
├── 9. 每日数据       ← 写作统计
├── 10. 全局搜索      ← 跨模块检索
├── 11. AI 写作助手   ← AI 辅助
├── 12. 资料库        ← 参考资料管理
├── 13. 导出系统      ← 多格式导出
├── 14. 作品管理      ← 多作品/系列管理
├── 15. 修订与编辑    ← 改写流程
├── 16. 写作日志      ← 反思与成长
├── 17. 投稿追踪      ← 出版流程
└── 18. 协作与同步    ← 多人协作
```

## 功能优先级定义

| 优先级 | 含义 | 阶段 |
|--------|------|------|
| P0 | 核心功能，MVP 必须 | Phase 1 |
| P1 | 重要功能，提升完整体验 | Phase 2 |
| P2 | 增值功能 | Phase 3 |
| P3 | 锦上添花 | Phase 4+ |

## 模块1: 正文写作 (P0)

### 1.1 卷管理
- 创建/编辑/删除/排序卷
- 字段：标题、副标题、序号、状态（规划/起草中/修订中/已完成）、概要、目标字数、实际字数（从章节聚合）
- 拖拽排序

### 1.2 章节管理
- CRUD + 排序
- 字段：章节编号、标题、状态、POV角色、字数目标、实际字数、摘要、标签、创建时间、修改时间
- 章节类型：序章/常规/间章/终章/附录/番外
- 章节锁定（防止误编辑）
- 章节备注（作者私密笔记）

### 1.3 场景管理
- 场景卡片：标题、摘要、字数、POV角色、地点、时间、情绪/基调、冲突类型、参与角色、状态
- 场景内拖拽排序
- 场景分隔符

### 1.4 富文本编辑器 (TipTap)
- 基础格式化：粗体/斜体/下划线/删除线/H1-H6/引用/列表/分割线
- 写作专用：场景分隔符插入、章节标题自动格式化、段落缩进控制、智能引号
- Markdown 支持 + 实时预览切换
- 自定义样式（字体、字号、行高、背景色、文字颜色）
- 全屏专注模式 + 打字机滚动
- 自动保存（默认30秒间隔）
- 字数统计：选中/当前场景/当前章/当前卷/全书
- 查找替换（支持正则）
- 阅读时间估算

### 1.5 专注模式
- 三级：简约UI / 打字机模式 / 禅模式
- 可配置背景（纯色/渐变/纹理/自上传）
- 环境音集成
- 番茄钟集成
- 会话目标（"写500字"或"写30分钟"）
- 会话统计

### 1.6 分屏/双栏
- 横/竖分屏
- 参考面板：可固定大纲/人物/世界观/参考资料
- 同步滚动（大纲与章节联动）

### 1.7 章节状态与标签
- 可配置状态流：大纲→初稿→自审→Beta→终稿
- 自定义标签 + 颜色标记
- 筛选/排序
- 进度条

### 1.8 导出 (Phase 4)
- 格式：TXT/DOCX/PDF/EPUB/MOBI/HTML/Markdown
- 范围：单章/卷/全书/自选
- 格式配置 + 导出预设

### 1.9 版本管理 (Phase 4)
- 自动快照 + 手动快照
- 版本对比（diff视图）
- 命名版本
- 回滚

### 1.10 批注/注释 (Phase 4)
- 行内注释（锚定到文本范围）
- 类型：自注/待审查/待调研/待重写/连续性检查
- 状态：开放/已解决/已忽略
- 注释仪表盘

## 模块2: 大纲设计 (P0)

### 2.1 故事结构模板
- 三幕式/英雄之旅(12步)/拯救猫咪(15拍)/七点式/丹哈蒙故事圈/四幕式(起承转合)/雪花法
- 自定义结构模板
- 每拍：名称/描述/建议位置(%进度)/关联章节
- 结构叠加可视化

### 2.2 大纲节拍
- 节拍卡片：标题/描述/结构阶段/目标字数/位置/情绪强度(1-10)/冲突等级(1-10)/状态/关联角色
- 拖拽排序
- 按情节线颜色编码
- 节拍→章节关联

### 2.3 情节线管理
- A-plot/B-plot/C-plot
- 每条线：名称/描述/类型/主要角色/状态/起始章/结束章/解决状态
- 情节线鸟瞰图
- 跨线依赖追踪

### 2.4 大纲视图
- 树形层级视图
- 看板/卡片视图
- 时间线视图
- 思维导图视图
- 表格/电子表格视图

### 2.5 冲突与节奏
- 冲突类型标记 + 强度曲线
- 节奏评分(1-10) + 曲线图
- 主题/母题追踪

## 模块3: 世界观设计 (P0基础 + P1扩展)

### 3.1 设定总览 (P0)
- 世界名称/类型/时代/技术水平/魔法普及度/基调/简介
- 封面图/情绪板

### 3.2 地点/地理 (P0)
- 层级：世界→大陆→区域→国家→省份→城市→区→具体地点
- 字段：名称/类型/父级/地图坐标/描述/人口/气候/建立日期/状态/控制势力/地标/动植物/进出口/知名居民(关联角色)/相关场景(关联章节)
- 地图标注（上传地图+标记点）
- 旅行路线 + 旅行时间计算器
- 秘境/特殊区域

### 3.3 势力组织 (P0)
- 字段：名称/全称/类型/徽标/格言/创立日期/创始人/当前领袖/总部/成员数/公开/秘密/立场/描述
- 组织架构（层级树）
- 成员名单（关联角色+职位+加入日期+忠诚度）
- 势力关系矩阵
- 资源与资产
- 势力范围（关联地点）

### 3.4 物品道具 (P0)
- 类型：武器/防具/工具/消耗品/神器/货币/载具/文件/服装/首饰/魔法物品/科技装置/食物/材料/关键物品
- 字段：名称/别名/类型/描述/外观/属性/等级(普通→神话)/价值/耐久度/制作要求/创造者
- 获取方式/当前归属(关联角色)/使用条件/诅咒/副作用
- 物品历史/传说
- 物品关系（套装/克制/材料链）

### 3.5 力量体系 (P1)
- 类型：硬魔法/软魔法/气/修炼/超能力/基因/科技/神赐
- 等级/境界（有序层级列表+能力解锁+条件+稀有度）
- 能力/技能目录 + 技能树
- 修炼方式/限制与代价/力量来源
- 相生相克矩阵

### 3.6 历史 (P1)
- 历史年表（与剧情时间线不同，这是背景历史）
- 重大事件：名称/日期/描述/原因/后果/关键人物/涉及势力/地点/死亡人数/持续影响
- 纪元/时代划分
- 传说/神话（含不同文化版本+真实版本）
- 战争史

### 3.7 扩展设定 (P2)
- 种族/物种：分类/特征/寿命/能力/文化/命名习惯
- 文化：习俗/礼仪/成年礼/婚葬/艺术/音乐/饮食/服饰/建筑
- 语言：语言谱系/文字系统/常用短语/命名规则
- 宗教/信仰：类型/神祇/经典/仪式/节日/教派
- 经济：货币/贸易/产业/阶级
- 政治：政体/法律/执法/外交
- 历法：自定义年月日/日期转换器
- 疾病/医疗/动植物/名言谚语

## 模块4: 人物设计 (P0)

### 字段清单
- **基本信息**：姓名/别名/性别/年龄(表面+实际)/生日/出生地/居住地/种族/国籍/职业/社会阶层/教育/语言
- **外貌**：身高/体型/肤色/发色发型/眼色/面部特征/特殊标志/惯用手/姿态/表情/声音/气味/着装风格
- **性格**：MBTI/九型人格(可选)/大五人格滑块/核心特质(5-7词)/美德(3-5)/缺陷(3-5)/道德阵营/气质/内外向
- **背景**：童年/家庭/关键成长事件/教育经历/职业经历/创伤/主要成就/秘密
- **动机目标**：目标描述/类型(短期/长期/隐藏)/优先级/起源/障碍/成功状态/内在动机vs外在目标/Want vs Need
- **能力技能**：名称/类别/熟练度/获取方式/关联力量体系能力
- **语言风格**：方言/词汇水平/句式复杂度/口头禅/语气词/幽默风格/音量/禁忌话题/称呼习惯/脏话习惯
- **习惯癖好**：紧张小动作/日常习惯/爱好/雷区/安慰物/睡眠/饮食/整洁度
- **角色弧光**：类型/起始状态/中点转折/最终状态/关键场景/主题关联
- **恐惧弱点**：恐惧描述/类型/强度/起源/表现/限制/能否克服
- **价值观**：核心价值/道德底线/宗教信仰/政治信念/偏见/底线
- **物品清单**：名称/关联物品条目/获取方式/意义

### 辅助工具
- 名称生成器（按文化/种族/性别/首字母/音节数/含义）
- 人物问卷（内置+自定义）
- 人物对比表（电子表格视图，行=人物，列=属性）
- 出场追踪（自动统计出现场景数/字数）
- 人物开发状态追踪

## 模块5: 人物关系图 (P1)

- 关系定义：角色A/角色B/类型(家人/恋人/社交/敌对/其他)/方向/强度(1-10)/状态/起止日期/相遇方式/公开认知vs真实/描述
- 可视化：React Flow 交互图，节点=角色(头像)，边=关系(颜色/粗细/箭头)
- 布局：力导向/层级/圆形/按阵营/按地点
- 筛选：按关系类型/阵营/重要性
- 关系变化时间线：选两个角色，看关系弧线
- 阵营可视化：按势力聚类，势力间连线
- 密度分析：枢纽角色/孤立角色/过度连接检测
- 多视角：客观视角 vs 各角色主观视角
- 导出：PNG/SVG/PDF

## 非功能性需求

### 性能
- 编辑器输入延迟 < 16ms
- 自动保存不阻塞 UI
- 章节列表 >500章时仍流畅（虚拟滚动）
- API 响应 < 200ms (P95)
- 首屏加载 < 3s

### 安全
- 所有 API 需 JWT 鉴权
- 密码 bcrypt 哈希
- SQL 注入防护（Prisma 参数化查询）
- XSS 防护（React 默认转义 + DOMPurify）
- CORS 配置
- 限流（rate limiting）

### 可访问性
- WCAG 2.1 AA 标准
- 键盘导航完整支持
- 屏幕阅读器友好
- 足够的颜色对比度

### 离线能力
- Service Worker 缓存编辑器核心
- 离线可新建/编辑章节
- 联网后自动同步
- 冲突处理策略
