### 遇到的问题
1. husky postinstall hook 阻止 npm install → 用 `--ignore-scripts` 跳过
2. Node.js 24 无法直接 require sharp → 先用 npm install 安装再用 require
3. PWA manifest 引用的 icon-192.png/icon-512.png 不存在 → 用 sharp 从 SVG 生成
4. 安卓端无法访问开发服务器 → Windows 防火墙默认拦截 5173 端口，需管理员权限添加放行规则：
   ```
   netsh advfirewall firewall add rule name="希陆Flow Dev Server" dir=in action=allow protocol=TCP localport=5173
   ```
   访问地址：`http://10.67.139.169:5173`（手机和电脑需在同一 WiFi 网络）
