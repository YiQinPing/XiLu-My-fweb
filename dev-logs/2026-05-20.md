## 功能架构总览

### 全局状态管理

| Store | 文件 | 用途 |
|-------|------|------|
| `useAuthStore` | `stores/auth.ts` | 用户认证状态、token 管理 |
| `useThemeStore` | `stores/theme.ts` | 主题切换（墨/纸/夜/森）、CSS 变量 |
| `useProjectStore` | `stores/project.ts` | 当前选中作品 ID，localStorage 持久化，跨页面自动同步 |

所有功能页面均使用 `useProjectStore` + `useSearchParams` 双保险获取 projectId：
```ts
const globalProjectId = useProjectStore((s) => s.selectedProjectId);
const projectId = globalProjectId || searchParams.get("project") || "";
```

### 共享组件

- **ProjectSelector** — 作品下拉选择器，自动读取全局 store，默认选中首个作品，支持"新建作品"入口
- **Sidebar** — 左侧导航，10 个功能入口 + 用户/主题设置

### 各功能页面详情

#### 1. Write 写作页面 (`pages/Write.tsx`)
- **布局**：左侧可折叠卷/章节目录树（280px）+ 右侧 TipTap 富文本编辑器
- **目录树**：显示所有卷（Volume），点击展开显示章节列表；每个卷 hover 显示"添加章节"、"重命名"、"删除"按钮
- **新建卷**：底部按钮 → 弹出模态框输入卷名
- **新建章节**：hover 卷名 → 点击 + 按钮 → 弹出 NewChapterModal（标题 + 编号 + 选卷）；或顶部"目录"面板中"新建章"按钮
- **重命名**：卷 hover 显示 Edit3 按钮 → 玻璃拟态重命名弹窗；章节标题点击内联编辑（Enter 保存 / Esc 取消 / blur 自动保存）
- **删除**：卷/章节均使用 ConfirmModal 玻璃拟态弹窗确认后删除
- **编辑器**：TipTap + StarterKit（支持 H1-H3）+ Placeholder 扩展
- **自动保存**：编辑后 2 秒 debounce 自动保存，Ctrl+S 手动保存
- **字数统计**：中文字数 + 英文单词数（混合统计）
- **API**：`chapter.ts` (getChapter/updateChapter/createChapter/deleteChapter), `volume.ts` (listVolumes/createVolume/updateVolume/deleteVolume)

#### 2. Outline 大纲页面 (`pages/Outline.tsx`)
- **布局**：左侧 300px 目录树 + 右侧全宽全高编辑面板
- **目录树**：`BeatTreeItem` 递归组件，支持展开/折叠（ChevronDown/Right）
- **类型系统**：ACT(幕) / SEQUENCE(序列) / SCENE(场景) / BEAT(节拍)，每种有独立颜色标记
- **选中高亮**：选中节拍左侧蓝色竖线（borderLeft）+ 背景高亮
- **添加幕**：顶部"添加幕"按钮 → prompt 输入标题 → 创建顶级 ACT 节拍
- **添加子节拍**：hover 目录项 → 点击 + 按钮 → prompt 输入标题
- **删除**：hover 显示删除按钮（Trash2），confirm 确认后级联删除子节拍
- **右侧编辑面板**：
  - 类型选择器：四个按钮切换 ACT/SEQUENCE/SCENE/BEAT，选中时填充颜色
  - 标题输入框：全宽文本输入
  - 描述文本域：flex-1 自动填满剩余空间，最小 320px
  - 操作栏：保存 / 取消 / 删除（红色）
- **故事模板**：Three-Act（三幕式）/ Hero's Journey（英雄之旅）/ Save the Cat / Seven-Point 等预设
- **API**：`outline.ts` (listOutline/createOutlineBeat/updateOutlineBeat/deleteOutlineBeat/applyTemplate)

#### 3. Characters 人物页面 (`pages/Characters.tsx`)
- **布局**：左侧 288px 人物列表 + 右侧全宽详情表单（7 个分区 Tab）
- **列表功能**：搜索过滤、按分组筛选、排序；显示头像首字 + 重要性点数
- **详情 Tab**：
  - 基本信息：姓名、性别、年龄、种族、职业、状态、重要性
  - 外貌：身高、体型、发色、瞳色、肤色、显著特征
  - 性格：MBTI、性格特质、优点、缺点、喜好、厌恶、恐惧
  - 背景：出生地、家族、经历、教育、秘密
  - 说话风格：口头禅、语气、语速、方言
  - 角色弧光：初始状态 → 转变 → 最终状态
  - 元数据：标签（TagInput）、关系概要
- **API**：`character.ts` (listCharacters/getCharacter/createCharacter/updateCharacter/deleteCharacter)

#### 4. World 世界观页面 (`pages/World.tsx`)
- **布局**：顶部标题 + 统计卡片 + 三 Tab（地点/势力/物品）
- **地点 Tab**：`LocNode` 递归树，支持父子层级（世界→大陆→国家→城市→地点），每节点可添加子地点/编辑/删除
- **势力 Tab**：卡片网格，显示组织名称、类型标签、描述
- **物品 Tab**：卡片网格，显示物品名称、类型标签、描述
- **新建/编辑**：模态框输入名称、类型、描述
- **API**：`world.ts` (listLocations/createLocation/updateLocation/deleteLocation, 同样 factions/items)

#### 5. Relationships 人物关系图 (`pages/Relationships.tsx`)

- **节点**：人物圆形节点（72px），自动网格排列
- **边**：smoothstep 曲线，按关系类型着色（家人#4ecdc4/爱情#ff6b6b/友情#4a9eff/敌对#c1554b/师徒#f0c75e/上下级#9b7ec4/其他#999），当前关系有动画效果，箭头标记

- **删除**：编辑弹窗内的删除按钮触发 ConfirmModal 确认

### 关键设计模式

1. **项目选择双重保障**：`useProjectStore`（global state + localStorage）+ `useSearchParams`（URL query）并用，确保跨页面导航不丢失
2. **分栏布局**：Write、Outline、Characters、Timeline 采用左栏列表 + 右栏详情的经典 WPS/Word 模式
3. **递归树组件**：Outline(BeatTreeItem)、World(LocNode) 使用递归渲染层级结构
4. **Tab 切换**：World(3 tabs)、AiAssistant(4 tabs)、Stats(2 tabs) 使用 Tab 栏组织多功能区域
5. **模态框编辑**：World、Timeline、Foreshadowing、Inspiration、Relationships 使用固定定位模态框进行创建/编辑
6. **响应式卡片网格**：Inspiration(sm:2 lg:3)、Foreshadowing 使用 CSS Grid 自适应布局
