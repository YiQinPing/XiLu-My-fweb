### 关键决策
1. **数据库暂用 SQLite**：PostgreSQL 未安装，开发阶段用 SQLite，后续平滑迁移（Prisma 适配器切换）
2. **bcrypt → bcryptjs**：Windows 环境缺少 Python 编译环境，改用纯 JS 版本
3. **PWA 优先而非原生应用**：一套代码覆盖双平台
4. **TipTap 作为编辑器**（已安装依赖，待 Step 3 集成）
5. **TypeScript 6.0 适配**：移除 deprecated `baseUrl`，使用新的 paths 解析

### 遇到的问题
1. npm init 不支持中文目录名 → 手写 package.json 解决
2. bcrypt 原生编译需要 Python → 改用 bcryptjs
3. Prisma schema 15 个验证错误 → 重写 schema，修复所有双向关系
4. PostgreSQL 未安装 → 改用 SQLite 本地文件数据库
5. TypeScript 6.0 `baseUrl` 已废弃 → 直接使用 paths
6. JWT 验证失败 → process.env 在模块顶层常量中读取时被缓存，改为函数惰性读取
7. .env 文件值带引号导致 JWT_SECRET 含双引号 → 移除引号
